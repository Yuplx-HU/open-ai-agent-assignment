# 评测基准规范

## 1. 问题格式

每个问题保存为一个独立的 `.json` 文件,包含以下字段:

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | string | 是 | 唯一标识,格式 `Q{NNNN}`,如 `Q0001` |
| `question` | string | 是 | 问题完整文本(选择题的选项嵌入此字段) |
| `type` | string | 是 | `short-answer` / `true-false` / `single-choice` / `multiple-choice` |
| `difficulty` | integer | 是 | 难度 1-10 |
| `answer` | object | 是 | 答案对象(见第2节) |
| `sources` | array | 是 | 答案来源(见第3节) |
| `theme` | array | 是 | 主题及关键词(见第3节) |

---

## 2. 答案对象

`answer` 是一个对象,包含 `type` 和 `content` 两个必填字段,并按题型追加扩展字段。

### 字段速查

| `answer.type` | 适用题型 | `answer.content` 类型 | 扩展字段 |
| --- | --- | --- | --- |
| `exact` | 判断题 | boolean | - |
| `exact` | 单选题 | string(如 `"A"`) | - |
| `exact` | 多选题 | array(如 `["A","C"]`) | - |
| `similar` | 简答题 | string | - |
| `keypoint` | 简答题 | string | `keypoints`(踩分点列表) |

### 踩分点格式(仅 `type: "keypoint"` 时)

```json
"keypoints": [
  {"point": "踩分点描述", "score": 分值}
]
```

---

## 3. 来源与主题

### 3.1 sources - 答案来源

```json
"sources": [
  {
    "knowledge_set_id": "知识集ID",
    "original_text": "引用原文片段"
  }
]
```

### 3.2 theme - 主题分类

列表,每个元素为 `{主题名: [关键词列表]}`,全部 kebab-case。

```json
"theme": [
  {"protein-corona": ["formation", "nanoparticle"]}
]
```

---

## 4. 完整示例

### 4.1 简答题 - 踩点给分

```json
{
  "id": "Q0001",
  "question": "在蛋白冠研究中,如何利用数据挖掘方法预测纳米颗粒的蛋白冠组成?",
  "type": "short-answer",
  "difficulty": 4,
  "answer": {
    "type": "keypoint",
    "content": "可通过收集纳米颗粒理化性质与蛋白冠组成的实验数据,利用特征选择筛选关键描述符,再构建分类或回归模型进行预测。",
    "keypoints": [
      {"point": "收集纳米颗粒理化性质与蛋白冠数据", "score": 3},
      {"point": "利用特征选择筛选关键描述符", "score": 3},
      {"point": "构建分类或回归模型进行预测", "score": 4}
    ]
  },
  "sources": [
    {
      "knowledge_set_id": "academic_001",
      "original_text": "Machine learning approaches for predicting protein corona composition based on nanoparticle physicochemical properties..."
    }
  ],
  "theme": [
    {"protein-corona": ["prediction", "nanoparticle"]},
    {"data-mining": ["feature-selection", "classification"]}
  ]
}
```

### 4.2 多选题

```json
{
  "id": "Q0005",
  "question": "以下哪些数据挖掘方法常用于基因组学数据分析?\nA. 差异表达基因筛选(统计检验)\nB. 聚类分析\nC. 蛋白质折叠预测\nD. 降维可视化(t-SNE/UMAP)",
  "type": "multiple-choice",
  "difficulty": 3,
  "answer": {
    "type": "exact",
    "content": ["A", "B", "D"]
  },
  "sources": [
    {
      "knowledge_set_id": "academic_002",
      "original_text": "Common data mining approaches in genomics include differential expression analysis, clustering, and dimensionality reduction..."
    }
  ],
  "theme": [
    {"genomics": ["differential-expression", "clustering"]},
    {"data-mining": ["statistical-testing", "dimensionality-reduction"]}
  ]
}
```

---

## 5. 文件组织

```
benchmark/
├── Q0001.json
├── Q0002.json
└── ...
```

> `knowledge_set_id` 应与知识集目录中的文件夹名称保持一致。

---

## 6. 数量要求

每组须构建**至少 20 个有效问题**。有效问题指能清晰回答、有准确来源、格式正确的问题。
