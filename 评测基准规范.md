# 评测基准规范

## 1. 问题格式

每个问题保存为一个独立的 `.json` 文件，包含以下字段：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | string | 是 | 唯一标识，格式 `Q{NNNN}`，如 `Q0001` |
| `question` | string | 是 | 问题完整文本（选择题的选项嵌入此字段） |
| `type` | string | 是 | `short-answer` / `true-false` / `single-choice` / `multiple-choice` |
| `difficulty` | integer | 是 | 难度 1–10 |
| `answer` | object | 是 | 答案对象（见第2节） |
| `sources` | array | 是 | 答案来源（见第3节） |
| `theme` | array | 是 | 主题及关键词（见第3节） |

---

## 2. 答案对象

`answer` 是一个对象，包含 `type` 和 `content` 两个必填字段，并按题型追加扩展字段。

### 字段速查

| `answer.type` | 适用题型 | `answer.content` 类型 | 扩展字段 |
| --- | --- | --- | --- |
| `exact` | 判断题 | boolean | — |
| `exact` | 单选题 | string（如 `"A"`） | — |
| `exact` | 多选题 | array（如 `["A","C"]`） | — |
| `similar` | 简答题 | string | — |
| `keypoint` | 简答题 | string | `keypoints`（踩分点列表） |

### 踩分点格式（仅 `type: "keypoint"` 时）

```json
"keypoints": [
  {"point": "踩分点描述", "score": 分值}
]
```

---

## 3. 来源与主题

### 3.1 sources — 答案来源

```json
"sources": [
  {
    "knowledge_set_id": "知识集ID",
    "original_text": "引用原文片段"
  }
]
```

### 3.2 theme — 主题分类

列表，每个元素为 `{主题名: [关键词列表]}`，全部 kebab-case。

```json
"theme": [
  {"protein-corona": ["formation", "nanoparticle"]}
]
```

---

## 4. 完整示例

### 4.1 简答题 — 踩点给分

```json
{
  "id": "Q0001",
  "question": "温室效应的主要原理是什么？",
  "type": "short-answer",
  "difficulty": 4,
  "answer": {
    "type": "keypoint",
    "content": "大气中的温室气体吸收地表发出的长波辐射并重新向地面发射，导致地表温度升高。",
    "keypoints": [
      {"point": "温室气体吸收长波辐射", "score": 3},
      {"point": "重新向地面发射", "score": 3},
      {"point": "地表温度升高", "score": 4}
    ]
  },
  "sources": [
    {
      "knowledge_set_id": "encyclopedia_001",
      "original_text": "Greenhouse gases absorb and re-emit infrared radiation from Earth's surface..."
    }
  ],
  "theme": [
    {"climate-science": ["greenhouse-effect", "radiative-forcing"]}
  ]
}
```

### 4.2 多选题

```json
{
  "id": "Q0005",
  "question": "以下哪些是可再生能源？\nA. 太阳能\nB. 天然气\nC. 风能\nD. 石油",
  "type": "multiple-choice",
  "difficulty": 3,
  "answer": {
    "type": "exact",
    "content": ["A", "C"]
  },
  "sources": [
    {
      "knowledge_set_id": "encyclopedia_002",
      "original_text": "Renewable energy sources include solar, wind, hydro..."
    }
  ],
  "theme": [
    {"general": ["renewable-energy", "solar-energy", "wind-energy"]}
  ]
}
```

---

## 5. 文件组织

```
benchmark/
├── Q0001.json
├── Q0002.json
└── ...
```

> `knowledge_set_id` 应与知识集目录中的文件夹名称保持一致。

---

## 6. 数量要求

每位学生须构建**至少 20 个有效问题**。有效问题指能清晰回答、有准确来源、格式正确的问题。
